{%- comment -%} hreflang tags. Resolves the EN/ES counterpart of the current page via: 1. `page.collection == 'posts'` paired by `pair_key` (most specific) 2. `page.collection == 'projects'` paired by `ref_pair` 3. `page.ref` matched against `_data/nav_manifest.yml` (fallback) Order matters: posts and projects inherit `ref: writing|work` from defaults, so collection checks must precede the nav_manifest fallback. Emits only when both counterparts resolve. x-default points to ES (site default language). {%- endcomment -%} {%- assign url_en = nil -%} {%- assign url_es = nil -%} {%- if page.collection == 'posts' and page.pair_key -%} {%- for p in site.posts -%} {%- if p.pair_key == page.pair_key -%} {%- if p.lang == 'en' -%}{%- assign url_en = p.url -%}{%- endif -%} {%- if p.lang == 'es' -%}{%- assign url_es = p.url -%}{%- endif -%} {%- endif -%} {%- endfor -%} {%- elsif page.collection == 'projects' and page.ref_pair -%} {%- for p in site.projects -%} {%- if p.ref_pair == page.ref_pair -%} {%- if p.lang == 'en' -%}{%- assign url_en = p.url -%}{%- endif -%} {%- if p.lang == 'es' -%}{%- assign url_es = p.url -%}{%- endif -%} {%- endif -%} {%- endfor -%} {%- elsif page.ref -%} {%- for item in site.data.nav_manifest.items -%} {%- if item.ref == page.ref -%} {%- assign url_en = item.en -%} {%- assign url_es = item.es -%} {%- endif -%} {%- endfor -%} {%- endif -%} {%- if url_en and url_es -%} {%- endif -%}
